## 해외 ATM 출금, 수수료를 모르면 손해

해외 여행이나 장기 체류 중 현금이 필요한 순간이 있습니다. 현지 ATM에서 카드로 현금을 출금하면 편리하지만, **생각보다 많은 수수료**가 부과될 수 있습니다. 어떤 카드를 쓰느냐, 어떤 ATM을 이용하느냐에 따라 같은 금액을 출금해도 수수료가 0원에서 수만 원까지 차이가 납니다.

이 글에서는 해외 ATM 출금 시 발생하는 수수료의 구조를 설명하고, 수수료를 최소화할 수 있는 카드를 비교 분석하겠습니다.

## 해외 ATM 출금 수수료 구조

해외 ATM에서 현금을 출금할 때 발생하는 수수료는 크게 세 가지입니다.

### 1. 카드사 해외 출금 수수료

한국 카드를 해외 ATM에서 사용할 때 카드사에서 부과하는 수수료입니다.

| 카드 유형 | 수수료 |
|---|---|
| 일반 체크카드 | 건당 3,000~5,000원 |
| 일반 신용카드 (캐시 어드밴스) | 건당 5,000원 + 이자 |
| 트래블월렛 | 0원 (일부 국가) |
| 토스 체크카드 | 건당 약 1,000~2,000원 |
| 카카오뱅크 체크카드 | 건당 약 1,000~2,000원 |

### 2. 환율 마진 (국제 브랜드 수수료)

Visa, Mastercard 등 국제 카드 네트워크에서 현지 통화를 원화로 변환할 때 적용하는 수수료입니다.

- **Visa**: 약 1%
- **Mastercard**: 약 1%
- **UnionPay**: 약 0.5~1%

### 3. 현지 ATM 수수료

현지 ATM 운영 은행에서 부과하는 수수료입니다. 이것은 한국 카드사와 무관하게 현지에서 부과됩니다.

| 국가 | ATM 수수료 | 비고 |
|---|---|---|
| 일본 | 110~220엔 (약 800~1,600원) | 세븐일레븐 ATM 무료인 경우도 |
| 미국 | $2~5 (약 2,700~6,800원) | 은행 ATM보다 독립 ATM이 비쌈 |
| 유럽 | 0~5유로 | 국가·은행별 상이 |
| 태국 | 220바트 (약 8,500원) | 매우 비쌈! |
| 필리핀 | 200~250페소 (약 4,600~5,750원) | 비쌈 |
| 호주 | 0~2AUD | 대부분 무료 |
| 영국 | 무료 (은행 ATM) | 독립 ATM은 1~2파운드 |
| 베트남 | 22,000~55,000동 (약 1,200~3,000원) | ATM별 상이 |

### 총 수수료 예시 (30만 원 상당 출금)

| 카드 | 카드사 수수료 | 환율 마진 | 현지 ATM | 총 수수료 | 비율 |
|---|---|---|---|---|---|
| 일반 체크카드 (태국) | 4,000원 | 3,000원 | 8,500원 | **15,500원** | 5.2% |
| 일반 체크카드 (일본) | 4,000원 | 3,000원 | 1,000원 | **8,000원** | 2.7% |
| 트래블월렛 (일본) | 0원 | 0원 | 1,000원 | **1,000원** | 0.3% |
| 토스 (일본) | 1,500원 | 3,000원 | 1,000원 | **5,500원** | 1.8% |

같은 30만 원을 출금해도 카드에 따라 **1,000원에서 15,500원**까지 차이가 납니다.

## 국제 직불카드 상세 비교

### 트래블월렛 (Travelwallet)

해외 ATM 출금에 가장 유리한 카드입니다.

| 항목 | 내용 |
|---|---|
| 발급 | 앱에서 무료 발급 |
| 지원 통화 | 40개 이상 |
| 해외 출금 수수료 | 충전 통화 출금 시 무료 |
| 환율 | 충전 시 환율 적용 (추가 마진 없음) |
| 1회 출금 한도 | 충전 잔액 범위 내 |
| ATM 수수료 | 현지 ATM 자체 수수료는 별도 |
| 국제 브랜드 | Visa / Mastercard |

**장점**:
- 해당 통화를 미리 충전하면 출금 수수료 0원
- 환율 유리할 때 미리 충전 가능
- 앱에서 실시간 잔액 확인

**단점**:
- 미리 충전해야 하므로 여유 자금 필요
- 충전하지 않은 통화는 자동 변환(수수료 발생 가능)

### 토스 체크카드

| 항목 | 내용 |
|---|---|
| 발급 | 토스 앱에서 신청 |
| 해외 출금 수수료 | 건당 약 1,000~2,000원 |
| 환율 | 토스 앱 내 환율 (기본 환율 + 약 1%) |
| 1회 출금 한도 | 약 100만 원 |
| 국제 브랜드 | Visa |

**장점**:
- 별도 충전 없이 토스 계좌 잔액으로 출금
- 실시간 환율 확인 가능
- 앱 사용 편리

**단점**:
- 건당 수수료 발생
- 환율이 트래블월렛보다 약간 불리

### 카카오뱅크 체크카드

| 항목 | 내용 |
|---|---|
| 발급 | 카카오뱅크 앱에서 신청 |
| 해외 출금 수수료 | 건당 약 1,000~2,000원 |
| 환율 | 카카오뱅크 환율 (기본 환율 + 약 1%) |
| 1회 출금 한도 | 약 100만 원 |
| 환율 우대 | 수시로 환율 우대 이벤트 |
| 국제 브랜드 | Mastercard |

**장점**:
- 카카오뱅크 계좌 직접 사용
- 환율 우대 이벤트 빈번
- 친숙한 앱 환경

**단점**:
- 건당 수수료 발생
- Mastercard이므로 일부 ATM에서 안 되는 경우 (드묾)

## 국가별 ATM 사용 가이드

### 일본

- **추천 ATM**: 세븐일레븐(7-Eleven), 우체국(Japan Post), 이온은행(AEON)
- **세븐일레븐 ATM**: 외국 카드 인식률이 가장 높음, 24시간 운영
- **수수료**: 세븐일레븐은 무료~110엔
- **1회 한도**: 보통 50,000~100,000엔
- **팁**: 일본어가 어려우면 영어(English) 버튼을 먼저 누르세요

### 유럽

- **추천 ATM**: 주요 은행(BNP, HSBC, Deutsche Bank 등) 내부 ATM
- **피해야 할 ATM**: Euronet, Travelex 등 독립 ATM (수수료 높음, DCC 강권)
- **수수료**: 은행 ATM은 보통 무료, 독립 ATM은 3~5유로
- **DCC 주의**: 반드시 "Without Conversion" 선택

### 동남아 (태국, 필리핀, 베트남)

- **태국**: ATM 수수료 220바트(약 8,500원)로 매우 비쌈. 한 번에 최대 금액 출금 추천
- **필리핀**: 200~250페소(약 4,600~5,750원). BDO, BPI ATM 추천
- **베트남**: BIDV, Vietcombank ATM 추천 (수수료 비교적 저렴)
- **팁**: 동남아는 ATM 수수료가 비싸므로, 환전소에서 현금 환전하는 것이 더 유리할 수 있음

### 미국

- **추천 ATM**: Chase, Bank of America, Wells Fargo 등 주요 은행 ATM
- **피해야 할 ATM**: 편의점, 바, 카지노 내 독립 ATM (수수료 $3~$8)
- **수수료**: 은행 ATM은 $0~$3, 독립 ATM은 $3~$8

### 호주

- **추천 ATM**: CBA, Westpac, ANZ, NAB의 은행 ATM
- **수수료**: 대부분 무료 (2017년부터 주요 은행 ATM 수수료 폐지)
- **팁**: 호주는 ATM 수수료가 거의 없어 카드 출금이 편리

## ATM 출금 시 주의사항

### 1. DCC 거부

ATM에서도 DCC가 적용될 수 있습니다. "Would you like to convert to Korean Won?"이라는 질문이 나오면 **반드시 "No" 또는 "Without Conversion"**을 선택하세요.

### 2. 카드 삼킴(Swallowed Card) 방지

- ATM 사용 후 카드를 빨리 빼지 않으면 보안상 카드를 삼킬 수 있습니다
- 카드를 빼라는 안내가 나오면 즉시 제거
- 카드가 삼켜지면 은행 영업시간에 방문하여 반환 요청

### 3. 스키밍 주의

- ATM 카드 투입구에 이상한 장치가 붙어 있는지 확인
- 키패드 위에 가짜 키패드가 덮여 있지 않은지 확인
- 가능하면 은행 건물 내부의 ATM을 이용
- 비밀번호 입력 시 손으로 가리기

### 4. 출금 한도 확인

| 카드 | 1일 해외 출금 한도 |
|---|---|
| 트래블월렛 | 충전 잔액 범위 내 |
| 토스 | 약 100만 원 |
| 카카오뱅크 | 약 100만~300만 원 (설정 변경 가능) |
| 일반 체크카드 | 약 100만~300만 원 |

출발 전 앱이나 은행에서 **해외 ATM 출금 한도**를 확인하고, 필요하면 한도를 조정하세요.

## 수수료 절약 팁 요약

1. **트래블월렛에 현지 통화 충전** → 출금 수수료 0원
2. **은행 내부 ATM 이용** → 현지 ATM 수수료 절약 + 안전
3. **한 번에 큰 금액 출금** → 건당 수수료 절약 (특히 태국, 필리핀)
4. **DCC 무조건 거부** → 3~8% 절약
5. **독립 ATM 피하기** → 높은 수수료와 DCC 회피
6. **출발 전 한도 확인** → 출금 실패 방지

### 카드별 추천 상황

| 상황 | 추천 카드 |
|---|---|
| 소액 수시 출금 | 토스, 카카오뱅크 (편리함) |
| 대금액 출금 | 트래블월렛 (수수료 0원) |
| 장기 체류 | 트래블월렛 + 현지 은행 계좌 |
| 비상용 | 일반 체크카드 (보조) |
