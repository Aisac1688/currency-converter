## 해외 프리랜서 수금, 어떤 방법이 가장 유리할까?

프로그래밍, 디자인, 번역, 컨설팅 등 다양한 분야에서 해외 클라이언트와 일하는 한국인 프리랜서가 늘고 있습니다. 해외에서 돈을 벌 때 가장 큰 고민 중 하나가 **"어떻게 받을 것인가?"**입니다.

수금 방법에 따라 **수수료가 2~7%까지 차이**가 나며, 1,000만 원을 받을 때 최대 70만 원까지 벌어질 수 있습니다. 이 글에서는 대표적인 3가지 수금 방법을 비용, 속도, 편의성 측면에서 철저히 비교합니다.

## 수금 방법 한눈에 비교

| 항목 | 페이팔 (PayPal) | Wise (구 TransferWise) | 은행 해외 송금 (SWIFT) |
|------|---------------|----------------------|---------------------|
| 수수료 | 3.49~4.99% | 0.4~1.0% | 건당 20~40달러 |
| 환율 마진 | 2.5~4.0% | 0.3~0.6% | 1.0~1.75% |
| 총 비용 (1,000달러 기준) | 약 60~90달러 | 약 7~16달러 | 약 35~55달러 |
| 입금 속도 | 즉시~1일 | 1~2영업일 | 2~5영업일 |
| 한국 원화 출금 | 가능 (수수료 별도) | 가능 (자동) | 자동 |
| 최소 수금액 | 1달러 | 1달러 | 제한 없음 (수수료 대비 비효율) |
| 클라이언트 편의성 | 매우 높음 | 높음 | 보통 |
| 세금 증빙 | 페이팔 명세서 | Wise 명세서 | 은행 입금 확인서 |

## 방법 1: 페이팔 (PayPal)

### 수수료 구조

페이팔은 전 세계에서 가장 많이 사용되는 온라인 결제 수단이지만, 수수료가 가장 비쌉니다.

| 수수료 항목 | 비율/금액 | 설명 |
|-----------|---------|------|
| 수취 수수료 (상업적 결제) | 3.49% + 0.49달러 | 클라이언트가 결제할 때 |
| 수취 수수료 (개인 송금) | 무료~2.99% | 국가에 따라 상이 |
| 환전 수수료 (달러→원화) | 2.5~4.0% | 페이팔 자체 환율 적용 |
| 한국 계좌 출금 | 건당 없음 (최소 잔액 조건 있음) | 최소 잔액 이상 시 무료 |

### 1,000달러 수금 시 실제 비용

| 단계 | 금액 | 비고 |
|------|------|------|
| 클라이언트 지급 | 1,000달러 | |
| 페이팔 수취 수수료 (3.49% + 0.49) | -35.39달러 | |
| 페이팔 잔액 | 964.61달러 | |
| 환전 (페이팔 환율, 시장 대비 약 3% 불리) | 약 1,263,000원 | 시장 환율 1,350원 기준 |
| 시장 환율로 직접 환전했을 경우 | 약 1,302,000원 | |
| **총 손실 (수수료 + 환율 마진)** | **약 87,000원 (6.4%)** | |

### 페이팔 비용 줄이는 팁

1. **페이팔 → Wise 연동**: 페이팔에서 달러를 Wise로 보내고 Wise에서 원화로 환전하면 환율 마진 절약
2. **달러로 유지**: 당장 원화가 필요 없으면 달러 잔액으로 보관, 환율 좋을 때 환전
3. **Invoice 활용**: "Send & Request" 기능으로 Invoice 발행 시 수수료 구조가 유리할 수 있음
4. **비즈니스 계정**: 매출 규모에 따라 수수료 협상 가능

### 페이팔이 적합한 경우

- 클라이언트가 페이팔만 사용하는 경우
- 소액 결제가 자주 발생하는 경우 (Upwork, Fiverr 등 플랫폼 연동)
- 빠른 수금이 중요한 경우

## 방법 2: Wise (구 TransferWise)

### 수수료 구조

Wise는 **실제 시장 환율(미드마켓 레이트)**을 적용하고, 별도의 투명한 수수료만 부과합니다.

| 수수료 항목 | 비율/금액 | 설명 |
|-----------|---------|------|
| 수취 수수료 | 무료 | Wise 계좌로 입금 시 |
| 환전 수수료 | 0.4~0.6% | 달러→원화 전환 시 |
| 한국 계좌 출금 | 무료~소액 | 자동 이체 |
| Wise 비즈니스 계좌 수수료 | 월 0달러 | 기본 기능 무료 |

### 1,000달러 수금 시 실제 비용

| 단계 | 금액 | 비고 |
|------|------|------|
| 클라이언트 Wise로 송금 | 1,000달러 | 송금 수수료는 클라이언트 부담 |
| Wise 수취 | 1,000달러 | 수취 수수료 무료 |
| 환전 수수료 (0.5%) | -5달러 | |
| 적용 환율 (시장 환율) | 1,350원 | |
| 원화 수령 | 약 1,343,250원 | |
| **총 비용** | **약 6,750원 (0.5%)** | |

페이팔 대비 약 **80,000원(12배) 저렴**합니다.

### Wise의 핵심 장점

1. **투명한 환율**: 은행간 실제 거래 환율(미드마켓 레이트) 적용
2. **멀티커런시 계좌**: 미국 달러, 유로, 영국 파운드 등 40개 통화 계좌 보유 가능
3. **로컬 계좌 번호**: 미국 ACH, 유럽 IBAN 등 현지 계좌번호 제공 → 클라이언트 입장에서 국내 송금처럼 간편
4. **Wise 비즈니스 카드**: 보유 외화로 직접 결제 가능
5. **자동 환전 설정**: 목표 환율 도달 시 자동 환전

### Wise 사용 시 주의사항

- **한국 계좌 등록 필수**: 본인 명의 한국 은행 계좌와 연결 필요
- **실명 인증**: 여권 등 신분증으로 본인 인증 필요
- **송금 한도**: 건당 100만 달러, 연간 한도는 상황에 따라 다름
- **일부 국가 제한**: 특정 국가에서 Wise 사용이 제한될 수 있음

## 방법 3: 은행 해외 송금 (SWIFT)

### 수수료 구조

전통적인 은행 간 국제 송금 방식인 SWIFT 네트워크를 통한 수금입니다.

| 수수료 항목 | 금액 | 설명 |
|-----------|------|------|
| 수취 수수료 (한국 은행) | 1~2만 원 | 은행마다 상이 |
| 중개 은행 수수료 | 10~25달러 | 경로에 따라 추가 |
| 환전 스프레드 | 1.0~1.75% | 은행 고시환율 적용 |
| 전신료 | 5,000~10,000원 | 일부 은행 면제 |

### 1,000달러 수금 시 실제 비용

| 단계 | 금액 | 비고 |
|------|------|------|
| 클라이언트 송금 | 1,000달러 | |
| 중개 은행 수수료 | -15달러 | 평균 |
| 수취 은행 도착 | 985달러 | |
| 수취 수수료 | -약 10달러 | |
| 환전 (은행 환율, 우대 50%) | 약 1,308,000원 | 스프레드 0.875% |
| **총 비용** | **약 42,000원 (3.1%)** | |

### 은행 송금이 적합한 경우

- 대금이 5,000달러 이상 대액인 경우 (고정 수수료 비중 낮아짐)
- 클라이언트가 기업이고 SWIFT 송금만 가능한 경우
- 세금 증빙이 명확해야 하는 경우 (은행 입금 내역이 가장 공식적)
- 정기적으로 같은 금액을 받는 경우 (환율 우대 협상 가능)

## 금액대별 최적 수금 방법

| 월 수금액 | 추천 방법 | 이유 |
|----------|---------|------|
| 100달러 이하 | 페이팔 | 편의성 우선, 수수료 절대액 작음 |
| 100~1,000달러 | Wise | 가장 저렴한 비용 |
| 1,000~5,000달러 | Wise | 비용 대비 효과 극대화 |
| 5,000~10,000달러 | Wise 또는 은행 송금 | Wise 환율이 유리하지만, 은행도 우대 협상 가능 |
| 10,000달러 이상 | 은행 송금 (우대 협상) | 고정 수수료 비중 낮음, 은행과 VIP 환율 협상 |

## 세금 신고: 놓치면 큰일

해외에서 수입을 받는 프리랜서도 한국에서 세금을 신고해야 합니다.

### 신고 의무

| 항목 | 내용 |
|------|------|
| 소득 유형 | 사업소득 (프리랜서) |
| 신고 주기 | 종합소득세 연 1회 (5월) |
| 부가가치세 | 해외 용역은 영세율 적용 (0%) |
| 환율 적용 기준 | 대금 수취일의 기준환율 또는 재정환율 |
| 필요 서류 | 수금 내역서, 계약서, Invoice |

### 세금 신고 팁

1. **모든 수금 기록 보관**: 페이팔/Wise/은행 명세서를 월별로 정리
2. **환율 기록**: 수금일자별 적용 환율을 기록 (국세청 고시환율 기준)
3. **경비 공제 활용**: 장비, 소프트웨어, 통신비, 작업 공간 등 필요경비 공제
4. **외화 계좌 관리**: 외화를 바로 환전하지 않아도, 수취 시점의 환율로 소득 인식
5. **세무사 상담**: 연간 수입이 2,000만 원 이상이면 세무사 활용 추천

### 주의사항: 해외 금융계좌 신고

해외 금융계좌(페이팔, Wise 포함) 잔액이 **매월 말일 중 하나라도 5억 원**을 초과하면, 국세청에 별도로 신고해야 합니다. 미신고 시 과태료가 부과됩니다.

## 환전 타이밍 전략

수금한 달러를 언제 원화로 환전하느냐에 따라 수익이 달라집니다.

### 전략 1: 즉시 환전

- 수금 즉시 원화로 환전
- 장점: 환율 리스크 제거, 현금 흐름 안정
- 단점: 환율이 유리한 시점을 놓칠 수 있음

### 전략 2: 목표 환율 환전

- 목표 환율을 설정하고, 도달 시 환전
- 장점: 더 유리한 환율에 환전 가능
- 단점: 목표에 도달하지 않으면 무기한 대기

### 전략 3: 분할 환전 (추천)

- 수금액의 일부는 즉시 환전, 나머지는 나중에 환전
- 장점: 환율 평균화 효과, 리스크 분산
- 단점: 관리가 다소 번거로움

### 실전 예시

매월 5,000달러를 수금하는 프리랜서의 분할 환전 전략:

| 시점 | 환전 비율 | 금액 | 이유 |
|------|---------|------|------|
| 수금 즉시 | 50% | 2,500달러 | 생활비 등 즉시 필요한 금액 |
| 월말 | 30% | 1,500달러 | 환율 추이 확인 후 환전 |
| 환율 유리 시 | 20% | 1,000달러 | 환율 알림 설정, 목표 도달 시 환전 |

## 플랫폼별 수금 방법

| 플랫폼 | 기본 수금 방법 | 추천 설정 |
|--------|-------------|---------|
| Upwork | 페이팔, Wise, 은행 | Wise 연결 (가장 저렴) |
| Fiverr | 페이팔, 은행 | 페이팔 → Wise 경유 |
| Toptal | 페이팔, Wise, 은행 | Wise 직접 연결 |

| GitHub Sponsors | Stripe → 은행 | 은행 직접 수취 |

## 마무리

해외 프리랜서의 수금 방법 선택은 수익에 직접적인 영향을 미칩니다. 대부분의 경우 **Wise가 가장 비용 효율적**이며, 특히 월 수금액이 100~5,000달러 범위일 때 그 효과가 극대화됩니다. 페이팔은 편의성이 높지만 비용이 가장 비싸고, 은행 송금은 대액 거래에 적합합니다.
